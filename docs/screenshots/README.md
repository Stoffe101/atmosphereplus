# Atmosphere+ Screenshot Guide

Use real in-game screenshots for GitHub and Modrinth. Do not add mockups or generated screenshots.

Recommended captures:

- `01-home.png` - Main menu / Home page with the Atmosphere+ panel open.
- `02-presets.png` - Presets page showing Favorite Presets, Preset Management, My Presets, Nether Presets, End Presets, and Prebuilt Presets where practical.
- `03-preset-pack-export.png` - Export Preset Pack UI with selected presets.
- `04-preset-pack-import.png` - Import Preset Pack UI showing a valid pack preview.
- `05-nether-before.png` and `06-nether-after.png` - Same Nether location before and after applying a Nether preset.
- `07-end-before.png` and `08-end-after.png` - Same End location before and after applying an End preset.
- `09-mood-overlay.png` - Mood Overlay / Color Grade controls.
- `10-biome-atmospheres.png` - Biome Atmospheres page.

Capture tips:

- Use the same camera angle for before/after pairs.
- Hide unrelated HUD clutter when possible.
- Avoid shader packs for the baseline screenshots unless the caption clearly says shaders are enabled.
- Include at least one screenshot showing `config/atmosphereplus-preset-packs/` import/export UI.
- Keep the final image names stable so README, Modrinth, and release notes can refer to them later.
